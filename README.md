# YouTube Clone

An exact, high-fidelity desktop replica of the YouTube interface built using React.js (Vite) and Material-UI (MUI).

## 🚀 Live Demo

Check out the live application here: **[Live Demo](https://joseph-francis42.github.io/youtube/)**

---

## ✨ Features

- **Collapsible Navigation Drawer:** Toggle sidebar between expanded explore sections and compact icon strip.
- **Top Category Tags Bar:** Horizontal scrollable category pill tags to instantly filter content.
- **Robust Video Player:** Native iframe player streaming directly from YouTube's privacy-enhanced embed endpoints with autoplay support.
- **Complete Settings Controls:** Accessible player speed, quality selection, annotations, and captions.
- **Responsive Layout:** Dynamic CSS grid layout optimized for mobile, tablet, and widescreen desktops.
- **Channel Details page:** Custom channel banner, subscriber/video statistics, and sub-tab panels (Videos, Playlists, About).
- **Interactive UI Elements:** Functional comments thread, description drawer, search capabilities, and subscription buttons.

---

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, Material-UI (MUI) v9, React Router v7
- **Styling:** CSS3, Emotion (Styled Components)
- **API Client:** Axios (for live integration and search requests)
